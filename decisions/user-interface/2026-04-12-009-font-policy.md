# Font Policy

## Context and Problem Statement

Scry2 adopted Beleren (UIDR headings) and the Mana icon font (UIDR-006 symbols) but had no formal policy for which font serves which role. Body text fell through to Tailwind's default system sans-serif with no intentional choice. MPlantin — the official MTG rules text font — was referenced in `mana.css` but never vendored. Adding MPlantin completes the MTG typographic palette and warrants a policy so fonts are used consistently.

## Decision

Vendor MPlantin (Regular + Italic) and establish a role-based font policy for all UI text.

### Font Inventory

| Role | Font | Utility class | Fallback chain | Where |
|------|------|---------------|----------------|-------|
| Display / headings | Beleren Bold | `.font-beleren` | serif | Page titles, section headings, stat card values, navbar logo |
| Labels / small caps | Beleren SmallCaps | `.font-beleren-sc` | serif | Badges, column headers, compact labels |
| Card rules text | MPlantin Regular | `.font-mplantin` | Palatino Linotype, Palatino, Georgia, serif | Card detail views, deck lists, anywhere MTG card text is rendered |
| Card flavor text | MPlantin Italic | `.font-mplantin italic` | (same chain, italic) | Flavor text — the italic variant is auto-selected by `font-style: italic` |
| UI body text | System sans-serif | (Tailwind default) | — | Navigation, controls, form labels, general prose |
| Console / code | Monospace stack | (existing in app.css) | JetBrains Mono, SF Mono, Menlo, Consolas, monospace | Console drawer, code blocks |
| MTG symbols | Mana icon font | `mana_symbol/1` component | — | Mana pips, card types, abilities (per UIDR-006) |

### Font Files

All fonts are vendored in `priv/static/fonts/`. No CDN, no npm.

| File | Size | Source |
|------|------|--------|
| `Beleren2016-Bold.ttf` | 137 KB | Delve Fonts (commissioned by WotC) |
| `Beleren2016SmallCaps-Bold.ttf` | 145 KB | Delve Fonts |
| `Beleren2016SmallCaps-BoldItalic.ttf` | 140 KB | Delve Fonts |
| `MPlantin.ttf` | 70 KB | Community distribution of WotC's MPlantin |
| `MPlantin-Italic.ttf` | 77 KB | Community distribution of WotC's MPlantin |
| `mana.woff2` / `mana.woff` | 187 KB / 408 KB | andrewgioia/Mana, SIL OFL 1.1 / MIT |

### Rules

1. **No raw `font-family` in templates.** Always use utility classes (`.font-beleren`, `.font-beleren-sc`, `.font-mplantin`).
2. **Beleren is for headings and values only.** It only ships Bold weight — using it for body text produces an uncomfortably heavy block of text.
3. **MPlantin is for MTG-authentic card content.** Rules text, flavor text, card names in detail views. Not for UI chrome — it's a serif designed for small card text, not for navigation or form labels.
4. **System sans-serif is the default.** General UI text stays on the system font stack. It's fast, readable, and keeps the MTG fonts feeling special rather than omnipresent.
5. **All fonts vendored.** Files pinned in `priv/static/fonts/`, referenced with absolute paths (`/fonts/...`) in CSS `@font-face` declarations. `font-display: swap` on all faces.

### CSS Location

All `@font-face` declarations and utility classes live at the top of `assets/css/app.css`, grouped by font family with a comment block explaining each family's role.

## Consequences

**Good:**
- Complete MTG typographic palette — headings (Beleren), card text (MPlantin), symbols (Mana), UI (system sans)
- Clear role boundaries prevent font misuse and keep the design coherent
- MPlantin Italic gives authentic flavor text rendering without a separate font family

**Neutral:**
- Adds ~147 KB to `priv/static/fonts/` (MPlantin Regular 70 KB + Italic 77 KB). Both are TTF; modern browsers fetch on first use and cache. Acceptable for an MTG domain application.

**Risk:**
- MPlantin and Beleren are WotC proprietary fonts distributed by the community. Scry2 is a self-hosted personal tool, not a commercial product — this is the same usage context as MSE, Cockatrice, and other community MTG tools.
